# Parking-application
 An android app for solving an issue:3
 # идея
 Като паркираш на синя зона, да избереш час и да цъкнеш един бутон. А приложението само да изпраща смс-и. Да не ти е на главата, че ще ти сложат скоба.
 # как работи
 Сложил съм foreground service ,който праща sms си на определен номер с данни, до определен час . Всички данни се запазват в базата.
 # архитектура 
 MVVM имам модели, база, репо, view модел.
 # потребителски поток
 # стъпки за стартиране
 При избиране на кола, парко място и време, до кето да е е актвино. Натисни start.
